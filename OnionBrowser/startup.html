Disclaimer: Using Onion Browser does not inherently guarantee security or privacy. Browsing through Tor is significantly slower than through a non-tunneled browser connection.

  • IMPORTANT: HTML5 <video> tags can leak queries and data transfers outside of Tor, therefore breaking anonymity.
  • Javascript cannot be disabled, HTML5 Geolocation API cannot be disabled.
