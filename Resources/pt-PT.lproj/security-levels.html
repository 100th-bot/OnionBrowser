Security Levels

Host security settings have been grouped in ways to give you an overall level of security while browsing sites. Below is a mapping of the settings in each level. They may alter performance in different ways.

1. Insecure

(Not recommended)

Tudo deve funcionar.

Warning: This could leak your information and potentially be dangerous. This level allows web pages to use some functions (WebRTC, Audio/Video, etc) that might connect over your original connection instead of tunneling through Tor.
