Niveles de Seguridad

Los niveles de seguridad del Host se han agrupado de manera que te dan un nivel general de seguridad cuando estas abriendo sitios web. Debajo hay un mapa de la configuracion de cada nivel. Estos niveles pueden alterar el desempeño en distintos modos.

1.No es seguro

( no es recomendable )

Todo debería funcionar.

Advertencia: Esto podria filtrar informacion y probablemente peligroso. Este nivel permite que los sitios web usen algunas funciones (WebRTC, Audio/Video, etc) Esto podria conectarte directamente a tu conecion de internet sin pasar por Tor.

Política de contenidos
Abrir (modo normal de navegación)
Protección de enlace universal
Inactivo
WebRTC
Habilitado
Permitir recursos de modo-mixto
Sí

2.Moderado

(Recomendado)

Algunas cosas no van a funcionar.

JavaScript esta permitido, pero algunas de sus funciones como WebSockets, geolocation, XHR, etc., siguen explícitamente deshabilitadas.

Política de contenidos
No hay conexiones XHR/WebSocket/Vídeo
Protección de enlace universal
Activo
WebRTC
Deshabilitado
Permitir recursos de modo-mixto
No

3. Seguro

(Recomendado)

Muchas cosas no van a funcionar

Deshabilita JavaScript, WebSockets, Geolocation, XHR, etc.

Política de contenidos
Estricta (sin JavaScript, vídeo, etc.)
Protección de enlace universal
Activo
WebRTC
Deshabilitado
Permitir recursos de modo-mixto
No
